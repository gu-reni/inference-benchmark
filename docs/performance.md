# 推理性能对比报告

**测试环境**：
- CPU: Intel Xeon (Ice Lake) 2.8GHz, 2 核
- 内存: 3.5GB
- 操作系统: Ubuntu 22.04
- C++ 版本: ONNX Runtime 1.15.1
- Python 版本: PyTorch 2.0 + Flask

**测试图片**：(600KB, 224x224)

| 指标 | C++ (ONNX Runtime) | Python (PyTorch) | 加速比 |
|------|-------------------|------------------|--------|
| 单次推理耗时 (ms) | **71.4** | **180.2** | **2.52x** |
| 内存占用 RSS (MB) | 120 | 450 | 3.75x |
| 首请求延迟 (ms) | 76 | 210 | 2.76x |
| 预热后 P99 (ms) | 73 | 185 | 2.53x |

**结论**：C++ 服务在延迟和内存效率上均显著优于 Python 版本，验证了 C++ 在 AI 推理场景下的工程价值。
